# CBIR - Content Based Image Retrieval System

## Overview
A deep learning-based Content Based Image Retrieval (CBIR) system built with Flask and ResNet50. Users can upload an image and retrieve visually similar images from the database.

## Research Results
- Precision@5: 85.7%
- Recall@5: 22.5%
- mAP: 73.7%

## Features
- User Login & Register
- Image Upload & Retrieval
- Search History
- Dark Mode
- Similarity Chart
- Image Comparison
- Mobile Friendly

## Dataset
- Animals-10 Dataset
- 500 images (50 per class)
- 10 categories: Dog, Horse, Elephant, Butterfly, Chicken, Cat, Cow, Sheep, Spider, Squirrel

## Technology Stack
- Python
- Flask
- PyTorch
- ResNet50
- SQLite
- HTML/CSS/JavaScript

## How to Run
1. Install Python 3.13
2. Install libraries: `pip install -r requirements.txt`
3. Run: `python flask_app.py`
4. Open: `http://127.0.0.1:5000`
